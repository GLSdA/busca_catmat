## 📦 Extração de Dados CATMAT – Compras.gov

Este projeto contém um script desenvolvido em Python com o objetivo de automatizar a extração da relação de códigos **CATMAT** disponibilizados nos **dados abertos do portal Compras.gov.br**.

O script realiza o consumo das bases públicas, permitindo obter de forma estruturada informações relacionadas a materiais cadastrados no sistema de compras governamentais. Essa extração pode ser utilizada como insumo para análises de dados, integração com outros sistemas ou construção de bases consolidadas voltadas à gestão e monitoramento de compras públicas.

### 🎯 Objetivo

Facilitar o acesso e tratamento dos dados de CATMAT, promovendo:

* Padronização na coleta de dados públicos
* Automação de processos repetitivos
* Apoio a análises voltadas à gestão orçamentária e compras governamentais

### ⚙️ Funcionalidades

* Download automatizado dos dados abertos do Compras.gov
* Extração e organização da relação de códigos CATMAT
* Preparação dos dados para uso em análises futuras

### 🧰 Tecnologias utilizadas

* Python 3.x
* Bibliotecas padrão para manipulação de dados e requisições

### 📊 Possíveis aplicações

* Análise de despesas públicas por tipo de material
* Apoio à elaboração de painéis e dashboards
* Integração com bases de execução orçamentária
* Estudos sobre padronização de compras no setor público
